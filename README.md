kgatlw
======

<!-- badges: start -->
<!-- badges: end -->
**kgatlw** are palettes inspired by King Gizzard and the Lizard Wizzard's artwork by [Jason Galea](https://www.jasongalea.com/).

Installation
------------

``` r
install.packages("devtools")
devtools::install_github("nlopezc/kgatlw")
```

Available palettes
------------------

``` r
library(kgatlw)
```
